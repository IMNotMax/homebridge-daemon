# homebridge-daemon
Script to run Homebridge as a daemon on OS X

##Installation


in a terminal :
> $ sudo ./install.bash

if this command fails:
> $ sudo sh install.bash


##Restarting Homebridge (if you modify the config.json file for example)

in a terminal :
> $ sudo ./restart.bash

if this command fails:
> $ sudo sh restart.bash
